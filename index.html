TuTurnoApp · libreta de estudio creativo

Una app de reservas creativas para talleres con alma.

TuTurnoApp es una app pensada para talleres creativos y espacios con actividades: cerámica, pintura, fotografía, costura, cocina, flores, lettering y más.

Sirve para organizar reservas, controlar plazas, gestionar pagos, vender bonos o tarjetas regalo y ofrecer una experiencia más clara tanto para el negocio como para quien reserva.

Nació personalizada para La Purísima Conchi, pero la base puede adaptarse y reutilizarse para otros talleres de cualquier tipo.

Qué es TuTurnoApp, explicado fácil

TuTurnoApp es una solución digital para talleres y negocios creativos que necesitan organizar reservas sin depender de mensajes, hojas sueltas o gestión manual. El usuario puede ver actividades, elegir fecha, reservar y pagar; y el negocio puede controlar plazas, consultar reservas y ordenar mejor su día a día.

No está pensada como una app cerrada para un único sitio, sino como una base reutilizable que puede adaptarse a distintos talleres. Cambiando contenidos, imágenes, estilo visual y reglas de reserva, puede convertirse en una app propia para cada marca.

Qué puede hacer

  • Reservas de clases y talleres
  • Compra de bonos y tarjetas regalo
  • Control de plazas disponibles
  • Perfil de usuario con historial
  • Panel para administración del taller
  • Base adaptable para otros negocios creativos

Quién está detrás

Soy Alicia Santofimia Melero, la persona que ha creado TuTurnoApp.

Me gustan los proyectos útiles, las cosas bonitas y el orden dentro del caos. TuTurnoApp es mi primera app y el comienzo de un camino en el que quiero seguir creando herramientas digitales pensadas para proyectos creativos y personas que hacen cosas con sus manos.

Me gusta pensar en las personas que van a usar lo que hago. Tengo una mezcla de curiosidad, mirada creativa y esa cabezonería buena que hace falta para empezar algo desde cero y seguir mejorándolo poco a poco hasta que todo encaja.

Mini resumen sincero

Una app para talleres creativos, hecha con cariño y pensada para evitar el típico momento de: “espera, ¿quién había reservado esta plaza?”

Por qué puede reutilizarse

La lógica principal ya sirve para muchos casos: usuarios, reservas, horarios, aforo, pagos, bonos y gestión. Luego se adapta el diseño, los textos, las clases y las reglas concretas de cada taller. Eso hace que TuTurnoApp no sea solo una app cerrada, sino una base sólida para personalizar según cada negocio.
