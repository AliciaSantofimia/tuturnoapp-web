TuTurnoApp es una app pensada para talleres creativos y espacios donde se organizan actividades como cerámica, pintura, fotografía, costura, cocina, arte floral, lettering y muchas más.

Funciona como una libreta digital de reservas: permite organizar plazas, gestionar pagos, ofrecer bonos o tarjetas regalo y mantener todo más claro tanto para el negocio como para quien reserva.

TuTurnoApp comenzó desarrollándose para el taller La Purísima Conchi, y a partir de esa experiencia se diseñó una base que puede adaptarse fácilmente a otros talleres y proyectos.

Qué es TuTurnoApp, explicado fácil

Es una app pensada para talleres y espacios donde se organizan actividades.

Permite gestionar reservas de forma sencilla, sin depender de mensajes, hojas sueltas o sistemas improvisados. Las personas pueden ver las actividades disponibles, elegir fecha y reservar fácilmente, mientras el taller mantiene el control de plazas y organiza mejor su día a día.

La app no está pensada para un único taller. Funciona como una base adaptable que puede personalizarse para distintos proyectos cambiando contenidos, imágenes y estilo.

Qué puede hacer

  • Reservas de clases y talleres
  • Gestión de bonos y tarjetas regalo
  • Control de plazas disponibles
  • Perfil de usuario con historial
  • Panel para administración del taller
  • Base adaptable para otros negocios creativos

Ideal para talleres donde...

Se organizan clases o actividades con plazas limitadas, horarios y personas que quieren reservar de forma clara y sencilla.

Quién está detrás

Soy Alicia Santofimia Melero, la persona que ha creado TuTurnoApp.

Me gustan los proyectos útiles, las cosas bonitas y el orden dentro del caos. TuTurnoApp nace de esa mezcla: una herramienta pensada para talleres y proyectos creativos que necesitan organizar sus reservas de forma clara y sencilla.

Es mi primera app y el comienzo de un camino en el que quiero seguir creando herramientas digitales útiles para proyectos creativos y personas que hacen cosas con sus manos.

Por qué puede reutilizarse

TuTurnoApp tiene una base que ya resuelve muchas de las necesidades habituales de un taller: usuarios, reservas, horarios, plazas o gestión de actividades.

A partir de ahí se pueden adaptar el diseño, los textos, las clases y las reglas de funcionamiento según cada proyecto.

Por eso no es solo una app pensada para un único taller, sino una base que puede personalizarse para distintos negocios creativos.
